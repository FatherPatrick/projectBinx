# App Icon Drop Zone

Put your source app icon image in this folder when ready.

Recommended file:

- icon-1024.png (1024x1024, square, PNG)

Suggested path:

- projectBinx/assets/app-icon/icon-1024.png

After adding the image:

1. Android Studio: generate launcher icons from this source image.
2. Xcode: fill AppIcon asset slots from this source image.

This folder is just the source-of-truth image location so the team always knows where the icon file lives.
